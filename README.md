
## SpaceX Falcon 9 Landing Prediction using Data Science
This repository contains data analysis and machine learning models for SpaceX rocket launches.

### Overview

This project analyzes SpaceX Falcon 9 launch data to identify factors that influence the successful landing of the first-stage booster. The goal is to build predictive models that estimate landing success, improving mission efficiency and reducing costs through booster reusability.

### Objectives
- Analyze historical launch data
- Identify key factors affecting landing success
- Perform data visualization and exploratory analysis
- Build and evaluate machine learning models for prediction

### Methodology
- Data Collection: SpaceX REST API & Wikipedia scraping
- Data Wrangling: Cleaning, preprocessing, feature engineering
- EDA: Visualization and SQL-based analysis
- Interactive Analysis: Folium maps & Plotly Dash dashboards
- Modeling: Classification algorithms with hyperparameter tuning
